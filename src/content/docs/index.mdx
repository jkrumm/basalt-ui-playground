## Welcome

This documentation covers the CBBI Blueprint project — a TanStack Start dashboard with Blueprint v6 components, MDX-powered content, and SSR-first architecture.

<Admonition type="tip" title="Getting started">
  Head to the [Getting Started](/docs/getting-started) section to set up the project.
</Admonition>

## What's included

- **CBBI Dashboard** — Real-time Bitcoin cycle indicator composite
- **Blueprint v6** — Palantir's UI component system for data-dense interfaces
- **TanStack Start** — SSR-first React meta-framework with file-based routing
- **MDX Content** — Blog and docs powered by MDX with Shiki syntax highlighting
