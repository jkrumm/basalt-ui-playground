## Getting started

This section walks you through setting up and running the CBBI Blueprint project.

<Steps>
  <Step title="Prerequisites">
    You need Bun ≥ 1.2 and Node.js ≥ 22.12.0 installed on your machine.
  </Step>
  <Step title="Clone the repository">
    ```bash git clone https://github.com/jkrumm/cbbi-blueprint.git cd cbbi-blueprint ```
  </Step>
  <Step title="Install dependencies">
    <PackageManagerTabs command="install" />
  </Step>
  <Step title="Start the development server">
    ```bash bun run dev ``` Open [http://localhost:3000](http://localhost:3000) to see the
    dashboard.
  </Step>
</Steps>
